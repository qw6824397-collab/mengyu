# 素材目录说明

建议后续在这里新增：

```text
assets/home/logo.png
assets/home/banner.png
assets/home/entry-ticket.png
assets/home/entry-booking.png
assets/home/entry-activity.png
assets/home/entry-scan.png
assets/home/store-cover.png
```

## 图片尺寸建议

- `logo.png`：透明 PNG，建议宽 360px，高 150px 内。
- `banner.png`：横版主视觉，建议 686px × 256px。
- 四个入口图：透明 PNG，建议 220px × 220px。
- `store-cover.png`：门店封面，建议 686px × 310px。

## 设计注意

- 不要使用 Nickelodeon、海绵宝宝、朵拉、汪汪队等现成版权角色。
- 入口图建议统一做成圆形 3D 角色徽章。
- Banner 要保留中间文字区，左右放原创角色和乐园元素。
